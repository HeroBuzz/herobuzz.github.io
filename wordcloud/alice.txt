A área de tecnologia é inclusiva?
Se você acompanhou a discussão no Twitter na semana passada, viu que deu pano prá manga, certo? 

Resumo rápido da nossa visão: não é!

Deixe-nos compartilhar um dado que acompanhamos e medimos no AWS User Group São Paulo, um grupo de usuários com mais de 8000 pessoas e que tem encontros regulares todos os meses a mais de 5 anos. Menos de 3% das pessoas que comparecem aos encontros do grupo são mulheres. 

De todas as propostas de atividades do AWS Community Day Brasil, 14% foram feitas por mulheres. 

O problema é ainda mais complicado, pois há outras dimensões. Quer dizer, estamos falando de gênero. O que acontece, por exemplo, quando a análise é feita levando-se em consideração pessoas de baixa renda? 

Se formos analisar os locais nos quais os encontros ocorrem (pensando em encontros presenciais), o grupo acontece longe das periferias e longe dos locais mais afastados da cidade de São Paulo. Ah, alguém pode argumentar, os encontros virtuais tornam o grupo mais inclusivo! Não é tão simples. Se considerarmos apenas a infraestrutura básica, estamos falando de acesso à internet e computador, por exemplo. Isso é uma barreira e tanto.  

Analise o quadro geral (economia/desemprego/acesso a educação), inclua aí aspectos como ter alguma noção de inglês, acesso a cursos e recursos para estudo. Nossa área está longe de ser inclusiva!
Novidades AWS
AWS Glue Crawlers com suporte para notificações de eventos do Amazon S3. Um Glue Crawler torna a descoberta de conjuntos de dados mais simples, lendo dados no Amazon S3, por exemplo, extraindo seu esquema e mantendo automaticamente o catálogo de dados e os metadados. Com este lançamento é possível configurar notificações de eventos do Amazon S3 como fonte para os crawlers atualizarem incrementalmente as tabelas do catálogo de dados do AWS Glue. Este novo modo vai proporcionar redução do custo e tempo para processamento. 

Edição em massa do OpsItems no AWS Systems Manager OpsCenter. Uma boa redução no trabalho pesado não diferenciado.Você pode editar tanto no console como com um documento de automação.

Calculadora de preços da AWS agora com suporte para o Amazon CloudFront. Toda ferramenta que nos ajuda a navegar na complexidade dos preços da AWS é bem-vinda. Só não entendemos como que isso não estava disponível antes.

Melhorias em tabelas no Amazon QuickSight.. Diversas pequenas e boas melhorias em tabelas (tanto tabelas comuns como dinâmicas) para quem usa o QuickSight. 

O AWS Fault Injection Simulator permite injetar falhas em instâncias spot. Ótimo lançamento para quem está praticando a engenharia do caos e usa o AWS Fault Injection Simulator.
